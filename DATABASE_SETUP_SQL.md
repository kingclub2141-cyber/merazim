# Supabase Database Setup

For a foolproof setup, copy the contents of the file **`setup.sql`** in this project and paste them into your Supabase SQL Editor.

## Steps:
1. Open your project in Supabase.
2. Click on **SQL Editor** in the left sidebar.
3. Create a **New Query**.
4. Paste the content from `setup.sql`.
5. Click **Run**.
